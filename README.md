# PROJETO HACKATON DE DADOS UFPR
Este projeto foi desenvolvido visando melhorar a visibilidade dos dados das respostas aos formulários gerais de avaliação das disciplanas e cursos da UFPR, antes tratados e visualizados de maneira mais simples em arquivos xlsx. A ideia do projeto foi trazer uma forma simples e organizada de visualizar os estatisticamente esses dados do que seriam vistos em planilhas.

## Tecnologias:
- Python
- Streamlit
- Pandas
- Numpy
- Altair

## Como rodar o projeto:
1. Primeiro, garanta que o Python esteja instalado na sua máquina. Para verificar, basta acessar o terminal (MacOS e Linux já costumam vir instalados) e digitar **__python --version__** (Recomendamos a versão 3.13.9, que foi a utilizada para o desenvolvimento do projeto).
2. Instalar as dependências para rodar o projeto. Para instalar as dependências, abrir o terminal na pasta em que estiverem os arquivos do projeto (especificamente o requirements) e digitar **__pip install -r requirements.txt__**.
3. Rodar o projeto. O projeto está rodando via streamlit atualmente, então basta abrir o terminal onde estiver o arquivo app.py, e digitar **__python -m streamlit run app.py__** e irá abrir uma guia no navegador com o sistema rodando.
